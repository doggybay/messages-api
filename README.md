# Messages API

## Routes

- Get All
```
http://localhost:8000/api/messages
```

- Get One
```
http://localhost:8000/api/messages/:id
```

- Post New Message
```
http://localhost:8000/api/messages
```

- Patch Message
```
http://localhost:8000/api/messages/:id
```

- Delete Message
```
http://localhost:8000/api/messages/:id
```
